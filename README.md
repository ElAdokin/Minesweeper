# Minesweeper Game

A browser-based Minesweeper game developed with Unity and deployable via WebGL technology.
For testing you can play it at https://nicolascasal.com

## ðŸ§© Building the Game

This project is set up for Unity. To build the WebGL version:

1.  **Install Unity Hub** and make sure you have a compatible version of Unity Hub and Unity Editor installed
2.  Open this repository in the Unity Hub or open the project folder in the Unity Editor
3.  Go to `File > Build Settings`
4.  Select `WebGL` as the target platform
5.  Configure your WebGL build settings (API, scripting backend, etc.)
6.  Click `Build and Run`

## ðŸŽ® How to Play

1.  **Start the Game:** Open the generated HTML file (or navigate to the URL provided after building) in your compatible web browser
2.  **Left-click** to reveal a cell
3.  **Click on Bomb Button and click on cell** to place/detect a bomb
4.  **Select Difficulty:** Choose number of bombs in options menu
5.  **Objective:** Flag all mines without detonating any
6.  **Numbers:** Each revealed cell shows the number of adjacent mines
7.  **Win:** Clear all non-mine cells
